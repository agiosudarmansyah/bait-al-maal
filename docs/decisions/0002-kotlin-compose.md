# Decision

Replace Flutter with Kotlin + Jetpack Compose.

## Context

Flutter was originally selected to support multiple platforms from a single UI codebase.

As the project evolved, deeper Android knowledge and tighter platform integration became higher priorities.

## Edict

Android will use Kotlin and Jetpack Compose while Rust remains the shared business layer.

## Consequences

Positive

* Native Android experience
* Better tooling
* Improved platform understanding

Negative

* Separate iOS UI in the future
